# Essentials

## Project setup

``` sh
pnpm install
```

``` sh
bun install
```

### Compiles and hot-reloads for development

``` sh
pnpm dev
```

``` sh
bun run dev
```

### Compiles and minifies for production

``` sh
pnpm build
```

``` sh
bun run build
```

### Lints and fixes files

``` sh
pnpm lint
```

``` sh
bun run lint
```
